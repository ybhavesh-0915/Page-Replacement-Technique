# Page Replacement Algorithm

A small simulation of page replacement algorithms built with Ink and React.

## Features

- Demonstrates page replacement algorithm behavior
- Built with `ink` for terminal UI
- Uses React components in `src/PageReplacement.jsx`

## Installation

```bash
npm install
```

## Development

```bash
npm run daemon
```

This command watches the `src` folder and recompiles files to `dist` using Babel.

## Project Structure

- `src/` – source code and React components
- `dist/` – compiled output
- `package.json` – project metadata and scripts
